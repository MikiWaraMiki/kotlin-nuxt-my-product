# ユーザー登録について

- ユーザーは以下の方法で登録することできる
  - twitter
  - google account
  - email/PW
- email/PW の場合は、初回登録を仮登録とし、メールアドレスの検証を必須とする
- ユーザは登録後に、本登録情報を入力する
  - 入力する情報は下記とする
    - 会社名
      - 任意
    - 名前
      - 必須
    - 連携アカウント
      - 任意
    - アカウント id
      - 必須

## 本登録時に必要な情報

- アイコン(任意)
- ユーザ名(必須)
- パスワード(email の場合のみ)
- 苗字(任意)
- 名前(任意)
- サイトブログ(任意)
